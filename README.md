# dockaction
# (Node.js + Docker + GitHub Actions CI/CD)

This project is a complete, hands-on guide to building a Node.js application, containerizing it with Docker, and setting up an automated CI/CD pipeline to build and push the image to Docker Hub using GitHub Actions.

## 📂 Project Architecture

The repository contains the following core files:
* **`app.js`**: The main Node.js server.
* **`config/config.yaml`**: The configuration file required by the app to boot up.
* **`Dockerfile`**: Instructions for Docker to package the application.
* **`.github/workflows/ci.yml`**: The automation script for GitHub Actions.
